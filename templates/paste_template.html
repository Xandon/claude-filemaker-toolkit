FileMaker Scripts

Click Copy XML on any card, then in FileMaker open the target script in Script Workspace and paste (⌘V / Ctrl+V). FileMaker 18+ materializes the steps directly — MBS plugin is not required.

Paste order matters. When scripts call each other, create them in the order shown so the cross-references resolve cleanly.

If FileMaker says “Steps were not pasted because they require objects that don’t exist”: the field, layout, or TO IDs embedded in the XML don’t match this copy of the solution. FileMaker falls back to name matching, so most near-misses still paste — but renamed TOs or missing layouts produce that error. Regenerate this file from a fresh DDR index if the schema has shifted.
